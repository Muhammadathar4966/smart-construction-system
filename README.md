# Smart Construction Management System

A modern web application for managing construction projects with AI-based material recommendations.

## 🚀 Features

* Role-based dashboards (Client, Contractor, Admin)
* Project creation and cost estimation
* AI-based material recommendation system
* Construction progress tracking (images/videos)
* Communication system

## 🛠️ Tech Stack

* React + Vite
* Tailwind CSS
* shadcn UI
* Recharts

## 📦 Installation

```bash
npm install
npm run dev
```

## 📌 Purpose

This project is developed as a Final Year Project (FYP) to digitize construction workflows and improve decision-making using AI.
